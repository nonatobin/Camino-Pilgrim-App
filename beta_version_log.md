# Camino Pilgrim App — Beta Version Log

> This file tracks every beta release so testers and developers know what changed and when.

## v0.1.0-beta — March 29, 2026

**Status:** Initial Beta Release  
**Deployed to:** Perplexity Hosting  
**Changelog:**

- Dashboard with countdown to April 30 Camino start date
- Training Plan generator (auto-builds progressive 5+ week plan based on fitness level)
- Walk Timer with distance estimation (4.8 km/h average)
- Quick Log for pre-completed walks
- Pomodoro Timer with stretch/eye-relief prompts
- Route Map showing all 6 stages (Baiona → Vigo → Redondela → Pontevedra → Cuntis → Santiago)
- "Long Day" badges on 5+ hour stages
- Lodging info per stage
- Family Leaderboard with 6-character share codes
- Copy-stats-to-clipboard functionality
- Badge system (Trail Star, Marathon Pilgrim, Pomodoro Master, Eye Saver, Flame Keeper, Santiago Bound)
- Settings page with dark mode toggle
- Browser-native Web Speech API greeting
- Designed for seniors: large touch targets, readable text, bottom navigation
- No authentication required — one-tap onboarding

**Known Issues:**
- None reported yet (awaiting beta tester feedback)

**Testers:**
- April 30 Camino group (9 people)
- June Camino neighbor (1 person)
- SCA Multimedia / Hiking / Lifelong Learning club members
